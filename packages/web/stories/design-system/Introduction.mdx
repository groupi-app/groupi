# Groupi Design System

Welcome to the Groupi Design System - a fun-first, dramatically rounded design language inspired by Duolingo and Discord.

## Design Principles

### 1. Dramatically Rounded

Everything is soft and approachable with generous border radii:

- **Buttons**: 16px radius
- **Cards**: 20px radius
- **Modals**: 24px radius
- **Badges**: Pill-shaped (9999px)

### 2. Bouncy Animations

Playful spring/bounce easing on all interactions using `cubic-bezier(0.34, 1.56, 0.64, 1)`.

### 3. Vibrant Colors

A rich purple primary with bright accent colors:

- **Primary**: Purple `hsl(285, 100%, 34%)`
- **Success**: Bright Green
- **Warning**: Warm Orange
- **Error**: Bright Red
- **Celebration**: Golden Yellow

### 4. Semantic Tokens

Three-layer token architecture:

1. **Primitives**: Raw values (colors, sizes)
2. **Semantic**: Purpose-based (bg-surface, text-heading)
3. **Component**: Component-specific (button-radius, card-padding)
